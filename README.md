## 執行程式碼
### Django (後端)
1. 先開一個Terminal。
2. 在backend文件夾，輸入`python3 manage.py runserver`。
### React (前端)
3. 另外再開一個Terminal。
4. 在frontend文件夾，輸入`npm run dev`。
